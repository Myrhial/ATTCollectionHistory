# AllTheThings Collection History

A simple World of Warcraft addon that tracks your collection history in AllTheThings (ATT).

* View the collection history in a window by click the addon compartment entry or with `/attch show`
* View the recorded history as chat output with `/attch` (defaults to `session` filter)
* Add the parameters `session`, `day`, `week`, or `month` to filter the results in chat. For the complete history, please use the window.
